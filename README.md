# Ansible Labs

This repository contains Ansible Playbooks for real-world DevOps automation.

## Playbooks

## 1. System_health.yml
- checks uptime
- cheks disk usage

## 2. Install_webserver.yml
- installs apache
- starts and enables service

## How to Run

--- bash

ansible-playbook -i inventory/hosts playbooks/system_health.yml
